## 模块列表

| 模块 | 描述 |
|------|------|
| **incremental_calibration** | 增量式状态估计和优化框架 |
| **aslam_splines** | B样条曲线的优化框架封装 |
| **bsplines** | 基础 B样条曲线库 |
| **ethz_apriltag2** | AprilTag 视觉标记检测 |
| **kalibr** | 顶层标定工具箱 |
