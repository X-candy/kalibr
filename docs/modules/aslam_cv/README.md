# aslam_cv 模块集群

> 计算机视觉与相机几何处理库

---

## 模块列表

| 模块 | 描述 |
|------|------|
| **aslam_cameras** | 相机几何模型和框架处理库 |
| **aslam_cameras_april** | AprilTag 校准目标支持库 |
| **aslam_cv_backend** | 视觉后端优化和设计变量管理库 |
| **aslam_cv_error_terms** | 重投影误差和视觉约束实现库 |
| **aslam_cv_python** | Python 绑定库 |
| **aslam_cv_backend_python** | 后端优化 Python 绑定 |
| **aslam_cv_serialization** | 序列化库 |
| **aslam_imgproc** | 图像处理和相机去畸变库 |
| **aslam_time** | 时间处理和时间戳管理库 |
